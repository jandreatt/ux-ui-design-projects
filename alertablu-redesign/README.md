# AlertaBlu UX/UI Redesign

This repository contains UX/UI design work developed as part of an academic course focused on User Experience, Interface Design, and Design Thinking.

The project proposes a redesign of the **AlertaBlu** platform, aiming to improve usability, visual clarity, and overall user experience through a user-centered design approach.

---

## Project Overview

The redesign was developed through a structured academic process, combining design fundamentals, user research, and UX principles.  
The repository documents the main deliverables produced throughout the course.

---

## Project Deliverables

### 1. User Research & Personas
- Problem understanding and analysis  
- Creation of user personas based on research activities  
- Identification of user needs and pain points  

### 2. Initial Layout
